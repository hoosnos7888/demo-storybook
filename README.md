## storybook ?

- UI 컴포넌트를 독립적인 환경에서 그려볼 수 있는 툴 중 하나이다.
- 단위테스트 : 컴포넌트만 브라우저 화면에 렌더링하고 다양한 상태에 따른 결과물을 살펴볼 수 있다.
- 독립적인 환경에서 사용할 수 있다.
- 컴포넌트의 재사용성이 뛰어나다.
